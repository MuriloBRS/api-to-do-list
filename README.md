1-Descrição do projeto:

Esse projeto é uma API simples feita com o Node.js para gerenciamento de tarefas. 

As funções da aplicação é:

*Criar tasks

*Listar tasks 

*Atualizar tasks 

*Deletar tasks

Cada task tem os seguintes atributos: 

ID: "Número identificador"

title: "Nome da task"

completed: "Status de conclusão (true/false)" 

2-Tecnologias utilizadas:

*Node.js 

*HTTP 

*Thunder client

*JS

3-Instalação:

*Clonar arquivos do GitHub 

-git clone 

4-Execução: 

*Iniciar servidor:

-node app.js 

*Servidor rodando em: - http://localhost:3000

5-Solução:

Completed foi incluído em taskModel, taskController e taskService.

Rota para GET: tasks/id adicionada em taskRoutes, busca de task por id em taskServices e em taskController
